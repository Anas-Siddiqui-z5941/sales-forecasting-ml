## 📘 Sales Forecasting & Customer Insights using Machine Learning

### Overview

This project analyzes retail sales data and builds a machine learning model to predict future sales and generate business insights.

### Tools Used

- Python
- Pandas, Numpy
- Matplotlib, Seaborn
- Scikit-learn

### Features
- Data Cleaning & Preprocessing
- Feature Engineering
- Exploratory Data Analysis (EDA)
- Machine Learning Models (Linear Regression & Random Forest)
- Model Evaluation (MAE, RMSE, R² Score)
- Feature Importance Analysis

### Key Insights

- Sales show seasonal patterns across months
- Weekend sales tend to be higher
- Customer demographics (age, gender) influence purchasing behavior
- High-value transactions significantly impact revenue  

### Conclusion

Random Forest model outperformed Linear Regression and provided better predictions for sales forecasting.
